## Migration Guide

### From Alpha 11 to Alpha 12

1. **Update Dependencies**
   ```bash
   npm install @heroui/react@3.0.0-alpha.12
   ```

2. **CSS Classes**
   - Component classes now follow BEM convention
   - Update custom styles to use new class names
   - Default sizes are now included in base classes

3. **Component Props**
   - Some prop names have been standardized
   - Check TypeScript errors for updated prop interfaces
   - Update component compositions if needed

### From v2 to v3

v3 is a complete rewrite with breaking changes. See our [Migration Guide](/docs/guides/migration) for detailed instructions.

## Release Schedule

HeroUI v3 follows semantic versioning and has a regular release cycle:

- **Alpha Releases** - Weekly during development
- **Beta Releases** - Monthly with feature freeze
- **Stable Releases** - Quarterly with full testing
