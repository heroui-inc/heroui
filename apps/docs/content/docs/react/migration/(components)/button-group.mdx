## Overview

The ButtonGroup component in HeroUI v3 maintains similar functionality to v2 but with enhanced prop inheritance and styling options. ButtonGroup allows grouping related buttons together with consistent styling and spacing.

## Structure Changes

In v2, `ButtonGroup` grouped buttons together:

```tsx
import { ButtonGroup, Button } from "@heroui/react";

export default function App() {
  return (
    <ButtonGroup>
      <Button>First</Button>
      <Button>Second</Button>
      <Button>Third</Button>
    </ButtonGroup>
  );
}
```

In v3, `ButtonGroup` has the same basic structure but with enhanced prop inheritance:

```tsx
import { ButtonGroup, Button } from "@heroui/react";

export default function App() {
  return (
    <ButtonGroup>
      <Button>First</Button>
      <Button>Second</Button>
      <Button>Third</Button>
    </ButtonGroup>
  );
}
```

## Key Changes

### 1. Import Path

**v2:** Imported from `@heroui/react`  
**v3:** Imported from `@heroui/react` (same package, but unified import)

### 2. Prop Inheritance

**v2:** Limited prop inheritance to child buttons  
**v3:** Enhanced prop inheritance - `size`, `variant`, `isDisabled`, and `fullWidth` props on ButtonGroup are automatically passed to child buttons

### 3. Hide Separator

**v3:** New `hideSeparator` prop to hide visual separators between buttons

### 4. Variant Support

**v3:** ButtonGroup now supports `variant` prop that applies to all child buttons

## Summary

- Enhanced prop inheritance - set `size`, `variant`, `isDisabled`, and `fullWidth` on ButtonGroup instead of individual buttons
- Improved consistency and reduced code duplication
