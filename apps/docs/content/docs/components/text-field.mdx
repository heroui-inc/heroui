### Anatomy

Import the compound parts to mix labels, inputs, descriptions, and errors.

```tsx
import {TextField} from '@heroui/react';

export default function Example() {
  return (
    <TextField isRequired>
      <TextField.Label>Display name</TextField.Label>
      <TextField.Input placeholder="Jane" />
      <TextField.Description>Visible to other team members.</TextField.Description>
      <TextField.Error />
    </TextField>
  );
}
```

## Styling

### Passing Tailwind CSS classes

```tsx
import {TextField} from '@heroui/react';

function CustomTextField() {
  return (
    <TextField className="gap-2 rounded-xl border border-border/60 bg-content1 p-4 shadow-sm">
      <TextField.Label className="text-sm font-semibold text-default-700">
        Project name
      </TextField.Label>
      <TextField.Input className="rounded-lg border border-border/60 bg-content1 px-3 py-2" />
      <TextField.Description className="text-xs text-default-500">
        Keep it short and memorable.
      </TextField.Description>
    </TextField>
  );
}
```

### Customizing the component classes

Target the shared BEM classes to centralize styling.

```css
@layer components {
  .text-field {
    @apply flex flex-col gap-2 rounded-xl border border-border bg-content1 p-4 shadow-sm;
  }

  .text-field__label {
    @apply text-sm font-semibold text-default-700;
  }

  .text-field__input {
    @apply rounded-lg border border-border/70 bg-content1 px-3 py-2 text-sm;
  }

  .text-field__textarea {
    @apply rounded-lg border border-border/70 bg-content1 px-3 py-2 text-sm leading-6;
  }

  .text-field__description {
    @apply text-xs text-default-500;
  }

  .text-field__error {
    @apply text-xs text-danger;
  }
}
```

### CSS Classes

- `.text-field` – Root container
- `.text-field__label` – Label slot
- `.text-field__input` – Input slot
- `.text-field__textarea` – TextArea slot
- `.text-field__description` – Description slot
- `.text-field__error` – Error message slot

### Interactive States

- **Focus Within**: `:focus-within` or `[data-focus-within="true"]` on the root (applied automatically)
- **Invalid**: `[data-invalid="true"]` or `[aria-invalid="true"]` hides the description and styles errors
- **Disabled**: `[data-disabled="true"]` or `[aria-disabled="true"]`
- **Read Only**: `[data-readonly="true"]` or `[aria-readonly="true"]`
- **Required**: `[data-required="true"]` or `[aria-required="true"]`

## API Reference

### TextField Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isRequired` | `boolean` | `false` | Marks the field as required. |
| `isInvalid` | `boolean` | `false` | Sets the validation state to invalid. |
| `isDisabled` | `boolean` | `false` | Disables the entire field. |
| `isReadOnly` | `boolean` | `false` | Makes the field non-editable but focusable. |
| `validationBehavior` | `"native" \| "aria"` | - | Controls how validation feedback is announced. |
| `validate` | `(value: string) => string \| null` | - | Custom validation callback returning an error string or `null`. |
| `defaultValue` | `string` | - | Uncontrolled initial value. |
| `value` | `string` | - | Controlled value. |
| `onChange` | `(value: string) => void` | - | Called with the next string value when the field changes. |
| `children` | `React.ReactNode \| (props: TextFieldRenderProps) => React.ReactNode` | - | Compound parts or render prop with field state. |
| `className` | `string` | - | Tailwind classes merged with the root styles. |
| `ref` | `React.Ref<HTMLDivElement>` | - | Ref forwarded to the root element. |
| `...TextFieldProps` | - | All additional props from `react-aria-components` `TextField`. |

### TextField.Label Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Tailwind classes merged with the label styles. |
| `children` | `React.ReactNode` | - | Label text or elements. |
| `ref` | `React.Ref<HTMLLabelElement>` | - | Ref forwarded to the label element. |
| `...LabelProps` | - | Inherits props from `@heroui/react` `Label`. |

### TextField.Input Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Tailwind classes merged with the input styles. |
| `type` | `string` | `"text"` | Input type attribute. |
| `value` | `string` | - | Controlled value. |
| `defaultValue` | `string` | - | Initial value for uncontrolled usage. |
| `onChange` | `(event: React.ChangeEvent<HTMLInputElement>) => void` | - | Change handler. |
| `isDisabled` | `boolean` | `false` | Disables the input. |
| `isReadOnly` | `boolean` | `false` | Makes the input read only. |
| `isInvalid` | `boolean` | `false` | Marks the input invalid. |
| `ref` | `React.Ref<HTMLInputElement>` | - | Ref forwarded to the input element. |
| `...InputProps` | - | All other props from `@heroui/react` `Input`. |

### TextField.TextArea Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Tailwind classes merged with the textarea styles. |
| `rows` | `number` | `3` | Preferred visible line count. |
| `value` | `string` | - | Controlled value. |
| `defaultValue` | `string` | - | Initial value for uncontrolled usage. |
| `onChange` | `(event: React.ChangeEvent<HTMLTextAreaElement>) => void` | - | Change handler. |
| `isDisabled` | `boolean` | `false` | Disables the textarea. |
| `isReadOnly` | `boolean` | `false` | Makes the textarea read only. |
| `isInvalid` | `boolean` | `false` | Marks the textarea invalid. |
| `ref` | `React.Ref<HTMLTextAreaElement>` | - | Ref forwarded to the textarea element. |
| `...TextAreaProps` | - | All other props from `@heroui/react` `TextArea`. |

### TextField.Description Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Tailwind classes for the helper text. |
| `children` | `React.ReactNode` | - | Description content. |
| `ref` | `React.Ref<HTMLElement>` | - | Ref forwarded to the description element. |
| `...DescriptionProps` | - | Additional props from `@heroui/react` `Description`. |

### TextField.Error Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Tailwind classes for the error message. |
| `children` | `React.ReactNode \| (props: {validationErrors: string[]}) => React.ReactNode` | - | Error text or render prop. |
| `ref` | `React.Ref<HTMLElement>` | - | Ref forwarded to the error element. |
| `...FieldErrorProps` | - | Additional props from `@heroui/react` `FieldError`. |

### TextFieldRenderProps

| Prop | Type | Description |
|------|------|-------------|
| `isDisabled` | `boolean` | Whether the field is disabled. |
| `isInvalid` | `boolean` | Whether the field is invalid. |
| `isReadOnly` | `boolean` | Whether the field is read only. |
| `isRequired` | `boolean` | Whether the field is required. |
| `isFocused` | `boolean` | Whether any child control is focused. |
| `isFocusVisible` | `boolean` | Whether focus is keyboard visible. |
| `validationState` | `"valid" \| "invalid"` | Current validation state. |
